一、SQL注入的平衡

首先，再进行sql注入时，我们要先推断出查询语句的基础语法。

例如：

'''
select username,password from users where id = '输入的内容';

'''
而输入的内容就是我们注入的部分。通常的例子为
localhost/sqli-labs/Less-1?id=1
id 后的内容就是我们输入的内容。根据上面的源代码，我们可以知道输入的内容在放入语句中运行时，会被一对‘号包裹，里面的语句便不会被当成语法运行，所以，我们的第一步便是绕过闭合。
但现实中通常不会这么简单，闭合的方法不一定为’，可以有一下多种：
 id=1
 id='1'
 id="1"
 id=(1)
 id=('1')
 id=("1")

 所以，我们要逐个尝试，从而得出目标的组合类型。

 之后，在知道目标查询的闭合方法后，我们还需要处理掉多余的符号，否则该条语句便不会执行。
 通常，有两种方法，一种是用注释，将后面所有的内容当作注释。另一种，就是认为制造闭合。
